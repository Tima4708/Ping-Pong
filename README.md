# Ping-Pong

Ping-Pong эта игра мультиплейер на одном пк,

Ping-Pong is a multiplayer game on one PC,
where you have to use tiles to hit the ball.

Управление,Controls.
RUS.
Правая плита - W|S
Левая плита - Стрелка вверх|Стрелка вниз
ENG.
Right Plate - W|S
Left Plate - Up Arrow|Down Arrow
